📋 GASセットアップ手順を見る
① Googleスプレッドシートを新規作成
sheets.google.com を開いて「空白のスプレッドシート」を作成してください。
タイトルは「労務管理チームツール DB」など任意で構いません。
② Apps Script エディタを開く
スプレッドシートのメニュー →「拡張機能」→「Apps Script」を選択します。
エディタが開いたら、左側の「コード.gs」をクリックして中身を全選択して削除します。
③ GASコードを貼り付ける
下のボタンでコードをコピーして Apps Script エディタに貼り付けてください。
貼り付け後、「💾 保存」ボタン（Ctrl+S）を押します。
④ ウェブアプリとしてデプロイ
Apps Script エディタ右上「デプロイ」→「新しいデプロイ」をクリック。
種類：ウェブアプリ
次のユーザーとして実行：自分
アクセスできるユーザー：全員（Googleアカウントを持つユーザー）
→ 「デプロイ」ボタンを押してURLをコピー
⑤ URLをここに貼り付けて保存
コピーしたURLを上の入力欄に貼り付けて「保存」→「接続テスト」で確認してください。
✅ 接続成功 が表示されれば設定完了です。あとは「🔄 同期」ボタンで2名のデータが共有されます。
